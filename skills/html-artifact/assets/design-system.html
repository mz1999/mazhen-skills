How to Use

Save this file as design-system.html in your project root. When asking Claude to generate a new HTML artifact, include it as reference: 

Using design-system.html as the visual reference,
create a single-file HTML artifact that [your goal].

Requirements:
- Copy the :root token block exactly
- Use .panel for card surfaces
- Use .t-h1 / .t-h2 / .t-body for typography
- Use .layout-tri for side-by-side comparisons
- Keep it responsive (mobile-friendly)
- No external dependencies

Claude will inherit the color palette, fonts, spacing, and component styles automatically.
